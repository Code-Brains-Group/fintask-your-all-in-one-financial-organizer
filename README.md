# FinTask

**Personal finance + tasks + applications, with group collaboration.**

FinTask is an all-in-one productivity app for people who want to track money, manage tasks, hunt for jobs/scholarships, and do it together with friends, family, or teammates.

🔗 **Live app:** https://fin-task-buddy.lovable.app

---

## ✨ Features

### 💰 Finance Tracking
- **Transactions** across multiple wallets (cash, bank, M-Pesa, etc.)
- **Budgets & plans** with auto-renewing periods and item-level tracking
- **Savings goals** by repository type (Bank, MMF, Chama, SACCO, M-Pesa…)
- **Recurring rules** that can align to your custom fiscal period
- **Reports** filtered by today / week / month / year / custom — based on *your* fiscal calendar
- **Custom fiscal periods** — set your own month-start day and year-start month

### ✅ Task Management
- List, **Kanban board**, and dashboard views
- **Subtasks** with auto-complete parent when all children are checked
- Date grouping (Overdue, Today, Tomorrow, This week, Later)
- Custom date filters, priorities, labels, planned cost
- **Calendar sync** (.ics download / Google Calendar)
- Reminder lead times

### 🎓 Application Tracker
- Track jobs, scholarships, grants, school admissions in one place
- Status pipeline (Saved → Applied → Interview → Offer → Accepted/Rejected)
- Deadlines, contacts, amounts, reminders

### 👥 Groups & Collaboration
- Create flexible groups ("Family", "Job Hunt 2026", "Roommates")
- **Shareable invite links** — no email setup needed
- Share tasks, applications, and savings goals with a group
- **Roles:** Owner 👑 / Admin 🛡️ / Member — admins manage invites and members
- **Group dashboard** with stats, recent activity, and what's coming up
- **Member profile drawer** — click any member to see their tasks, apps, and contributions

### 🏆 Savings Showdown (Leaderboard)
- Turn group savings into a friendly competition
- Live leaderboard with 🥇🥈🥉 medals
- 🔥 Daily streaks for consistent contributors
- Deposit counts, last-deposit timestamps
- Shared progress bar toward combined targets

### 🎛️ Modules & Focus Mode
- Pick which modules you want at signup (Finance, Tasks, Applications) — change anytime in Settings
- **Focus mode** toggles between Finance-only / Tasks-only / Both views

### 🆘 In-app Help & Tour
- Guided tour for first-time users
- Module-by-module walkthroughs available from the Help page

## 🛠 Tech Stack

- **Frontend:** React 18 + Vite 5 + TypeScript 5

- **State / Data:** React Query, React Router
- **Backend:** Lovable Cloud (Supabase) — Postgres, Auth, Edge Functions, Row-Level Security
- **Charts:** Recharts
- **Drag & Drop:** dnd-kit
- **Dates:** date-fns
- **Icons:** lucide-react

## 🗄️ Data Model (high-level)

| Table | Purpose |
|---|---|
| `profiles` | User prefs, modules, fiscal config, notification settings |
| `wallets`, `transactions`, `categories` | Core finance ledger |
| `budgets`, `budget_items` | Budgets with auto-renew & purchasable items |
| `savings_goals`, `savings_contributions` | Goals + deposits (personal or group) |
| `recurring_rules`, `pending_recurring` | Recurring transactions with approval queue |
| `tasks`, `subtasks`, `task_assignees` | Tasks with hierarchy and assignment |
| `applications` | Jobs / scholarships / admissions pipeline |
| `groups`, `group_members`, `group_invites` | Collaboration primitives |

All user-data tables are protected by **Row-Level Security**. Group-shared rows check membership via the `is_group_member` / `is_group_admin` security-definer helpers.

## 🚀 Getting Started (local dev)

```bash
# Install deps
bun install   # or: npm install

# Run dev server
bun run dev   # or: npm run dev
```

The Supabase URL and publishable key are loaded from `.env` (auto-generated by Lovable Cloud).

## 🧭 How to use it (quick tour)

1. **Sign up** → tick the modules you want (Finance / Tasks / Applications)
2. **Set your fiscal calendar** in Settings if your month doesn't start on the 1st
3. **Create a group** (e.g. "Family Savings 2026") and copy the invite link
4. Share the link — anyone with it can join from `/join/CODE`
5. Open the group → see live **Savings Showdown leaderboard**, shared tasks, and applications
6. Click any member to see exactly what they're working on and contributing

---

## 🔒 Security Notes

- Email/password auth, no email confirmation required for fast onboarding
- Roles are stored on `group_members` (never on `profiles`) to prevent privilege escalation
- All cross-user data access flows through `SECURITY DEFINER` helpers with explicit `search_path`
- Sensitive operations (role changes, invite creation, member removal) are gated by RLS policies that call `is_group_admin`
